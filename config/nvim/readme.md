# Install

## install deno
```sh
curl -fsSL https://deno.land/x/install/install.sh | sh
mv ~/.deno/bin/deno /usr/local/bin
```

## install nerd font
```sh
wget https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/Hack.zip
unzip Hack.zip
sudo mv Hack.zip /usr/share/fonts/
sudo fc-cache -xvf
```
